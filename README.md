# ART API

## Getting Started

### Requirements

* Ruby 3.1.2+
* Rails 7.0.0+

### Run locally 

* Clone the repo
```
git clone https://github.com/Bright-Phiri/ART-Backend.git
```

### Setting up rails & other dependencies

```
cd ART-Backend
bundle install
```
Get everything up and running

### 3rd parties

ART API wouldn't be possible without:

* Twilio
* Cloudnary
